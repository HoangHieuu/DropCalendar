# Stories

Stories are bounded work packets linked to product contracts and executable
proof. Completed stories are:

- `epics/E00-foundation/US-000-wire-snapcal-spec/` — decomposed `SPEC.md`, equipped
  Harness, and leave an implementation-ready contract without scaffolding app
  code.
- `epics/E01-core-prototype/US-001-macos-screenshot-to-draft.md` — macOS image
  import, local Vision OCR, typed draft extraction, and editable review.

Future candidates are listed in `backlog.md`. They are not active until feature
intake classifies them and a durable Harness row exists.

Use `docs/templates/story.md` for normal work and
`docs/templates/high-risk-story/` for auth, privacy, external providers, data
retention/deletion, public contracts, or cross-platform work.
